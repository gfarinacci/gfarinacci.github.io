Jupyter and Hadoop

In order to run Jupyter you should install it on the server:

pip install jupyter --user

This command will work even if you donâ€™t have root access, installing Jupyter in your home directory rather than in /usr/local/lib/python.

Then in order to use the IDE there are two ways:

  • Using a remote display protocol
  • Making an SSH Tunnel

Of course I prefer the second option:

ssh -L 8888:127.0.0.1:8888 user@server_address

Then you can run the IDE:

python -m notebook --ip=127.0.0.1 --port=8888

This allows you to use Jupyter on your browser. Of course you should change the default port.
