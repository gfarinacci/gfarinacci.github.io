Demographic Indicator

The following R function let you download demographic indicators from the Eurostat website and save them in a dataframe with ease.

You can download and install the function directly from GitHub:

library(devtools)
install_github("ropengov/eurostat")
install_github("hadley/rvest")
install_github("gfarinacci/lifeTables")
library(lifeTables)

This library allows you to download the following indicators: Age specific death rate, Life expectancy at given exact age, Probability of dying between ages, Probability of surviving between exact ages, Person-years lived between exact age, Number left alive at given exact age, Total person-years lived above given exact age. For example if we want to plot the person-years lived between exact age for males, females and the general population in Italy in 2015:

library(lifeTables)
m <- lifeTable(geo = "IT", sex = "M", time = 2015, indic_de = "Lx")
f <- lifeTable(geo = "IT", sex = "F", time = 2015, indic_de = "Lx")
t <- lifeTable(geo = "IT", sex = "T", time = 2015, indic_de = "Lx")
m <- m[-86,]
names(m) <- c("age_m", "Lx Male")
f <- f[-86,]
names(f) <- c("age_f", "Lx Female")
t <- t[-86,]
names(t) <- c("Age", "Lx Total")
t <- cbind(t,m,f)
df<-t[,c("Age","Lx Total","Lx Male","Lx Female")]
remove(f,m,t)
require(ggplot2)
require(reshape2)
df <- melt(df ,  id.vars = 'Age', variable.name = 'Series')
names(df) <- c("Age", "Series", "Lx")
ggplot(df, aes(Age,Lx)) + geom_line(aes(colour = Series))
